2025 Utah Congressional Districts Plan

##Redistricting Data Hub (RDH) Retrieval Date
11/12/2025

##Sources
The shapefile and reports were retrieved from the [Campaign Legal Center](https://campaignlegal.org/document/plaintiffs-map-submission)

##Processing
The RDH did not modify any of the retrieved data. The block assignment file (BAF) was created by the RDH using the listed shapefile and the 2020 Census Block shapefile and the [maup library](https://github.com/mggg/maup).

##Additional Notes
Enclosed in this zip file the shapefile file for Utah's Congressional Districts (Plaintiff_Map_1.shp) and supporting files and BAF (ut_cong_adopted_2025_baf.csv).

To keep track of a state's adopted maps, go to All About Redistricting httpsredistricting.lls.edu
For any additional questions, please email info@redistrictingdatahub.or

Enacted by Utah's Third District Court on November 10, 2025.
